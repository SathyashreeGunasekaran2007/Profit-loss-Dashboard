# 📊 Profit and Loss Dashboard (Power BI)

## 📌 Overview
This project is a Power BI dashboard designed to analyze business performance using key metrics like revenue, profit, and expenses.

## 📈 Key Features
- 💰 Total Revenue Tracking
- 📊 Gross Profit & Net Profit Analysis
- 📉 Revenue YoY%
- 📅 Monthly Revenue Trends
- 🧾 Expense Breakdown by Category

## 🛠 Tools Used
- Power BI
- Data Visualization
- DAX (Data Analysis Expressions)

## 🔍 Insights
- Identified trends in revenue growth
- Analyzed expense categories impacting profit
- Compared monthly performance effectively

## 📁 Project File
- `sathyaproject.pbix`

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Explore the dashboard visuals
